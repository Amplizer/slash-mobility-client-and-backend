 - FrontEnd realizado en Angular ( Versión 7.3.10 )
 - Backend realizado en Spring e Hibernate ( Versión de Java 14 )
 - Motor de bases de datos: Mysql ( Versión 10.1.38-MariaDB )

	
 Para que el proyecto funcione, antes de nada se deberá ejecutar el script que se encuentra en la carpeta Mysql ( script.sql ). Con esto
 generarás la estructura de tablas necesaria para que la aplicación funcione.
 
 Luego deberás generar las dependencias Maven en el proyecto Spring, e iniciarlo

 Por último tienes que inicializar el front con el comando: ng s
